Detailed Description

template<typename T>
class Point< T >

A standard container for a point made up of an X and Y coordinate.

This Point is a simple class that contains an X and Y coordinate. It contains getters and setters for the coordinates and defines assignment, equality, inequality, and arithmetic operators for itself.

Template Parameters
TA generic paramter used to represent the data type used to represent the coordinates.
